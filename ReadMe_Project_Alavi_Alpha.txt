Project Alavi 
Integrantes: Anna Melendez, Jhony Atertua, Juan Camilo lasso, Miguel Rico
Es un juego que se enfoca en resolver unos puzzle para avanzar a niveles posteriores.
Actualmente el proyecto cuenta con tres niveles
1. La idea de este nivel es moverte por un laberinto para obtener unos puntos rojos, aquí tienes la oportunidad de utilizar botas cohete e impulsar al jugador ha plataformas ;y así abrir la puerta al siguiente nivel
   1. Para moverte se debe utilizar W y S para avanzar o retroceder, A y D para girar
   2. Para utilizar las botas cohete se debe presionar SPACE
   3. Para obtener los puntos solo debes tocarlos
   4. Al obtener todos los puntos se abrirá una puerta, lo que corresponde a ganar el nivel
2. Como segundo nivel debes organizar las piezas de tal manera en que sean iguales a la línea blanca que está en el piso
   1. Para moverte se debe utilizar W y S para avanzar o retroceder, A y D para girar
   2. Hay 3 botones, de color amarillo, morado y azul, te debes parar enfrente de esos botones y presionar E para que las piezas se muevan
   3. Si todas las piezas encajan el patrón del piso , se abrira una puerta que permitirá pasar al siguiente nivel
3. Este nivel busca que el jugador agarre objetos y los coloque en su color correspondiente, habrán unos ocultos y otros en plataformas por lo tanto deberá usar las botas cohete, cuando complete esta tarea se abrirá la puerta al nivel ganador.
   1. Para moverte se debe utilizar W y S para avanzar o retroceder, A y D para girar
   2. Para utilizar las Botas Cohete presionar SPACE
   3. Para agarrar los objetos utilizar la tecla E
   4. Debes acomodar todos los objetos para pasar a la siguiente pantalla
